# Internal Packages

This directory contains internal packages and configurations used by the monorepo.

## Content

- `lint-configs`: ESLint, Prettier, Stylelint, and Commitlint configurations.
- `node-utils`: Node.js utilities for scripts and build tools.
- `tailwind-config`: Tailwind CSS configuration presets.
- `tsconfig`: Shared TypeScript configurations.
- `vite-config`: Shared Vite configurations.
